사용방법

              
    
                
  1. 첫 화면에서 Mode를 선택합니다. Ubuntu 운영 제어는 Mode 1, Windows/Firebase 조회 중심은 Mode 2입니다.
    2. 
                
  2. Mode 1에서는 테스트 실행, 펌웨어 업로드, Watcher 제어를 사용합니다.
    3. 
                
  3. 결과서/업로드 메뉴에서 XML/HTML 결과서를 분석하고 저장 후 외부 업로드를 실행합니다.
    4. 
                
  4. Firebase 센터에서 Firestore 상태를 확인하고 동기화를 수행합니다.
    5. 
                
  5. 설정 메뉴에서 API Base URL을 저장하면 Hosting 화면에서도 Ubuntu API를 호출할 수 있습니다.
    6. 
              

            

          


          

            

              
도움말

              

                
Q. API 주소 저장은 언제 쓰나요?

                
Firebase Hosting 또는 Windows 브라우저에서 접속했을 때, Ubuntu 서버 API 주소를 고정해 호출하기 위해 사용합니다.

              

              

                
Q. 인터넷 상태/연결 상태는 어디서 보나요?

                
좌측 사이드 패널의 상태 카드에서 실시간으로 확인할 수 있습니다.

              

              

                
Q. 로그창이 화면을 가릴 때는?

                
하단 콘솔에서 최소화 버튼을 누르거나 상단 경계선을 드래그해 높이를 줄이면 됩니다.
